# Local Signal Scanner — Prototype v0.2

Adds an AI Answer / Generative Search Audit section for:
- ChatGPT Search
- Google Gemini
- Perplexity
- Microsoft Copilot
- Claude Web Search
- Grok / X

This remains a manual/live-demo prototype. It generates prompt packs, links to each AI platform, captures evidence notes, applies traffic-light status, and rolls failures into the priority fix plan.

Owner/admin verification, direct API checks, and automated AI answer collection are future integrations.
